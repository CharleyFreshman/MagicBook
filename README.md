# 英语魔法书

一个萌萌的、二次元风格的英语词组学习网页应用，基于语料库"Scopus.csv"设计，帮助用户轻松记忆英语词组。

## 功能特点

- 📚 随机抽取词库中的中文释义进行学习
- 🌐 支持中英文双语界面切换（完全本地化，无混合显示）
- 🎯 可选择测验题数（10、20、50、100题）
- 👁️ "瞄一眼"功能，逐步提示答案，增强记忆效果
- 📝 自动记录错题，便于针对性复习
- ✉️ 提供语料库纠错、分享新题库和用户体验优化的反馈渠道
- 📱 响应式设计，支持移动端和桌面端使用

## 技术栈

- HTML5
- CSS3 (二次元萌风格设计)
- JavaScript (ES6+)
- Vite (构建工具)
- XLSX (Excel/CSV文件解析)
- Chart.js (数据可视化)

## 安装与运行

1. 确保已安装 [Node.js](https://nodejs.org/)

2. 克隆项目到本地
   ```bash
   git clone https://github.com/CharleyFreshman/MagicBook.git
   cd MagicBook
   ```

3. 安装依赖
   ```bash
   npm install
   ```

4. 启动开发服务器
   ```bash
   npm run dev
   ```

5. 构建生产版本
   ```bash
   npm run build
   ```

6. 预览生产版本
   ```bash
   npm run preview
   ```

## 使用说明

1. 打开应用后，默认显示中文界面，可在顶部切换为英文界面
2. 选择测验题数，点击"开始测验"进入测验环节
3. 左侧显示中文释义，右侧为英文答案区域（优化字体，提高可读性）
4. 使用"瞄一眼"逐步查看答案，或使用"揭晓答案"直接查看完整答案
5. 确认答案后，点击"我答对了！"或"不认识"进入下一题
6. 完成所有题目后，查看得分和错题本
7. 可选择返回主界面或重新测验

## 语料库说明

- 项目默认使用"Scopus.csv"作为基础语料库（包含800多个常用英语词组）
- 每行左列为英文词组，右列为中文解释
- 同一单词的不同词组保持在同一行，通过换行符分隔
- 同一释义的多种表达组合在一起，显示时会保留原始格式
- 支持上传自定义词汇表文件（XLSX、XLS、CSV格式）
